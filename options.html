Backnote

Make a notes on a fly. Backnote window stays on top and let you copy in paste without using any keys, just by moving cursor over. Build a sheet from each session and save it as text file in one click. Backnote is fast and easy way to improve your productivity and save time. ★ Minimalistic One window that always stays on top of current page. ★ Easy-to-use Quick mouse movements instead of cascade of hotkeys and clicks. ★ Fast Light and simple substitution for a notepad in your browser.

✔ Open backnote window. ✔ Highlight text you want to copy. ✔ Hover over backnote window to paste it. ✔ When done making notes, click 'Save'

To open a backnote tab you can use hotkeys (Ctrl+Shift+V by default) or by clicking icon in upper-right corner of your browser. Hotkeys can be changed in a extension settings. After clicking 'Save', notes file will appear in your default downloads folder as 'backnote.txt' This extension is free and doesn't contain any ads.
